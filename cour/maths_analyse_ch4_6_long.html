4. Primitives et intégrales

Le calcul intégral prolonge la dérivation. Il permet de calculer des aires, des volumes, des longueurs et résout des problèmes inverses : retrouver une quantité à partir de sa vitesse. En analyse, toute fonction continue admet des primitives.

4.1 Primitives

Définition : Une fonction F est une primitive de f sur un intervalle I si F est dérivable sur I et pour tout x ∈ I, F '(x) = f(x). Deux primitives d'une même fonction diffèrent d'une constante réelle.

La recherche de primitive est l'opération inverse de la dérivation. Connaissant les dérivées usuelles, on en déduit des primitives.

4.1.2 Primitives et opérations

Si F et G sont des primitives de f et g, alors F+G est une primitive de f+g, et kF est une primitive de kf (k constante).

Primitives de fonctions composées : On utilise la formule (v∘u)' = u' · (v'∘u). Ainsi, pour reconnaître une forme u' × f(u), une primitive est F∘u où F' = f.

Exemples fondamentaux (tous les jours au Bac) :

  • ∫ u'(x) [u(x)]n dx = [u(x)]n+1/(n+1) + C, n ≠ –1.
  • ∫ u'(x) / u(x) dx = ln |u(x)| + C (u(x) ≠ 0).
  • ∫ u'(x) eu(x) dx = eu(x) + C.
  • ∫ u'(x) sin[u(x)] dx = –cos[u(x)] + C.
  • ∫ u'(x) cos[u(x)] dx = sin[u(x)] + C.

Remarque : Toute fonction continue admet des primitives, mais toutes ne s'expriment pas avec des fonctions élémentaires (exemple : e–x²).

4.2 Intégrale définie

Soit f continue sur [a,b]. L'intégrale de a à b de f(x) dx est la différence des valeurs d'une primitive quelconque : ∫ab f(x) dx = [F(x)]ab = F(b) – F(a).

Interprétation géométrique : Si f ≥ 0, l'intégrale représente l'aire du domaine situé entre la courbe, l'axe des abscisses et les droites x=a, x=b, exprimée en unités d'aire (u.a.). Pour une fonction de signe variable, l'intégrale est une aire algébrique (positive au-dessus, négative en dessous).

4.2.1 Propriétés de l'intégrale

  • Linéarité : ∫ (α f(x) + β g(x)) dx = α ∫ f(x) dx + β ∫ g(x) dx.
  • Relation de Chasles : ∫ab f = ∫ac f + ∫cb f, pour tout c.
  • Positivité : Si f ≥ 0 alors ∫ f ≥ 0. Si f ≥ g alors ∫ f ≥ ∫ g.
  • Valeur moyenne : μ = 1/(b–a) ∫ab f(x) dx. C'est la hauteur d'un rectangle de même aire.

4.3 Techniques d'intégration

4.3.1 Intégration par parties (IPP)

Formule : ∫ab u(x) v'(x) dx = [u(x) v(x)]ab – ∫ab u'(x) v(x) dx.

On choisit u et v' de manière à simplifier le calcul. On retient le moyen mnémotechnique : « dériver u, intégrer v' ».

4.3.2 Changement de variable (hors programme strict, mais peut être évoqué)

On pose x = φ(t), dx = φ'(t) dt, et on remplace les bornes.

4.4 Calcul d'aires entre deux courbes

L'aire du domaine compris entre les courbes de f et g (f ≥ g) sur [a,b] est ∫ab (f(x) – g(x)) dx.

Exercice 1 (Primitives usuelles) : Trouver une primitive de f(x) = 4x3 – 2/x + e2x.

F(x) = x4 – 2 ln |x| + (1/2) e2x + C.

Exercice 2 (IPP) : Calculer I = ∫0π x cos x dx.

Posons u = x ⇒ u' = 1 ; v' = cos x ⇒ v = sin x. Alors I = [x sin x]0π – ∫0π sin x dx = (π·0 – 0) – [–cos x]0π = 0 + (cos π – cos 0) = –1 – 1 = –2.

Exercice 3 (Aire) : Soit f(x)=x²–4x+5 et g(x)=2x–3. Déterminer l'aire entre les deux courbes entre leurs points d'intersection.

Points d'intersection : x²–4x+5 = 2x–3 ⇒ x²–6x+8=0 ⇒ x=2, x=4. Sur [2,4], f(x) ≤ g(x) ? On vérifie : en 3, f=2, g=3 → g≥f. Aire = ∫24 (g–f) = ∫24 (–x²+6x–8) dx = [–x³/3 + 3x² – 8x]24 = ( –64/3 + 48 – 32 ) – ( –8/3 + 12 – 16 ) = (16/3) – (–4/3) = 20/3 u.a.

5. Équations différentielles

Une équation différentielle est une relation entre une fonction inconnue et ses dérivées. Résoudre une telle équation, c'est trouver toutes les fonctions la vérifiant. En Terminale D, on étudie les équations linéaires du premier ordre.

5.1 Équation différentielle linéaire du premier ordre sans second membre : y' = a y

Forme générale : y' – a y = 0, où a est une constante réelle.
Solutions : Les fonctions y(x) = C ea x, C ∈ ℝ. La constante est déterminée par une condition initiale.

Exemple : y' = 3y → y = C e3x.

5.2 Équation avec second membre constant : y' = a y + b

On cherche une solution particulière constante y0 = k. En reportant : a k + b = 0 ⇒ k = –b/a (si a ≠ 0). La solution générale est la somme de la solution homogène et de cette solution particulière : y(x) = C ea x – b/a.

5.3 Équation avec second membre quelconque : y' = a y + f(x)

La méthode de variation de la constante consiste à chercher y(x) = C(x) ea x. En dérivant : y' = C' ea x + a C ea x. On remplace dans l'équation : C' ea x + a C ea x = a C ea x + f(x) ⇒ C' ea x = f(x) ⇒ C'(x) = f(x) e–a x.
On intègre pour obtenir C(x) = ∫ f(x) e–a x dx + K, puis la solution générale : y(x) = ( ∫ f(x) e–a x dx + K ) ea x.

Cette méthode est systématique mais parfois lourde ; on peut aussi chercher une solution particulière de la même forme que f(x) (polynôme, exponentielle, trigonométrique).

5.4 Problème de Cauchy

On ajoute une condition initiale du type y(x0) = y0. Cela fixe la constante d'intégration et assure l'unicité de la solution (sous réserve de bonnes hypothèses).

5.5 Applications : refroidissement, désintégration radioactive

La loi de Newton : T'(t) = –k (T(t) – Text). En posant θ = T – Text, on se ramène à θ' = –k θ. La solution exponentielle décrit le refroidissement.

Exercice 4 : Résoudre y' + 2y = 4, avec y(0)=1.

Équation homogène : yh = C e–2x. Solution particulière constante : yp = 2. Solution générale : y = C e–2x + 2. Condition : 1 = C + 2 ⇒ C = –1. Donc y = 2 – e–2x.

Exercice 5 : Résoudre y' – y = 3 e2x, y(0)=0.

Solution homogène : yh = C ex. Variation de la constante : C'(x) ex = 3 e2x ⇒ C'(x) = 3 ex ⇒ C(x) = 3 ex + K. Solution : y = (3 ex + K) ex = 3 e2x + K ex. y(0)=3+K=0 ⇒ K= –3. Finalement y = 3 e2x – 3 ex.

Exercice 6 : Une population croît selon le modèle P'(t) = 0.02 P(t) (t en années). En 2020, la population est 100000. Exprimer P(t) et calculer la population en 2030.

P(t) = P(0) e0.02t. On prend t=0 en 2020 → P(0)=100000. Donc P(t)=100000 e0.02t. En 2030, t=10 → P≈100000 e0.2 ≈ 122140.

6. Suites numériques

Les suites modélisent des phénomènes discrets : population d'une ville année par année, capital avec intérêts, etc.

6.1 Définitions et modes de génération

Suite explicite : un = f(n), n ∈ ℕ (ex : un = 1/n).
Suite récurrente : un+1 = f(un) avec u0 donné. La fonction f doit être définie sur un intervalle stable.

6.2 Suites arithmétiques et géométriques

Arithmétique : raison r. un = u0 + n r. Somme des n premiers termes (de 0 à n–1) : S = n (u0 + un–1)/2.

Géométrique : raison q. un = u0 qn. Somme : 1 + q + … + qn = (1 – qn+1)/(1 – q) si q≠1. Pour une suite géométrique, la somme des termes de 0 à n est u0 (1 – qn+1)/(1 – q).

6.3 Convergence des suites

La suite (un) converge vers L (fini) si tout intervalle ouvert contenant L contient tous les un à partir d'un certain rang. On écrit lim un = L. Si la suite ne converge pas, elle diverge (vers ±∞ ou sans limite).

Opérations : Les théorèmes sur les limites de sommes, produits, quotients de suites sont analogues à ceux des fonctions.

Théorèmes de comparaison : Si à partir d'un certain rang, vn ≤ un ≤ wn et que vn et wn convergent vers la même limite L, alors un → L.
Si un ≥ vn et vn → +∞, alors un → +∞.
Si un ≤ wn et wn → –∞, alors un → –∞.

Convergence monotone : Toute suite croissante et majorée converge. Toute suite décroissante et minorée converge. C'est un outil puissant pour les suites récurrentes.

6.4 Suites récurrentes un+1 = f(un)

On suppose f continue sur un intervalle I stable par f (i.e. si x ∈ I, f(x) ∈ I). Si la suite converge vers L, alors L doit être un point fixe de f : L = f(L).

Étude de la monotonie : On peut examiner le signe de un+1 – un = f(un) – un. Si f est croissante, la suite est monotone (croissante si u1 ≥ u0, décroissante sinon).

Représentation graphique « en toile d'araignée » ou « en escalier » : on trace la droite y = x et la courbe de f, puis on projette.

6.5 Suites adjacentes

Deux suites (an) et (bn) sont adjacentes si l'une est croissante, l'autre décroissante et leur différence tend vers 0. Alors elles convergent vers la même limite. Exemple : approximation de √2 par dichotomie.

Exercice 7 : Soit (un) définie par u0=5 et un+1 = 0.5 un + 1. Montrer que (un) converge et trouver sa limite.

Point fixe L = 0.5L + 1 ⇒ L = 2. On pose vn = un – 2. Alors vn+1 = 0.5 vn, suite géométrique de raison 1/2 → 0. Donc un → 2.

Exercice 8 : Étudier la convergence de un = (3n + 2n) / 5n.

un = (3/5)n + (2/5)n. Chacun tend vers 0 car |3/5|<1 et |2/5|<1. Donc lim un = 0.

Exercice 9 : Soit u0=1 et un+1 = 1 + 1/un. Montrer que la suite est bien définie, monotone ? (plus délicat). Mais on demande de montrer que si elle converge, sa limite est le nombre d'or φ = (1+√5)/2.

Si lim un = L, alors L = 1 + 1/L ⇒ L² – L – 1 = 0 ⇒ L = (1±√5)/2 ; comme un > 0, L = (1+√5)/2.
