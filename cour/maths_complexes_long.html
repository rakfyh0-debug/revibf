NOMBRES COMPLEXES – Cours complet
Terminale D – Algèbre et Géométrie

1. Introduction et définition

Les nombres complexes apparaissent historiquement pour résoudre les équations du troisième degré (Cardan, Bombelli, XVIe siècle). Ils forment un ensemble noté ℂ, extension de ℝ, dans lequel toute équation polynomiale a une solution (théorème de d'Alembert-Gauss, admis). On introduit un nombre imaginaire i tel que i² = –1.

Forme algébrique : tout nombre complexe s'écrit de manière unique z = x + i y avec x et y réels.
x = partie réelle Re(z), y = partie imaginaire Im(z). Si y = 0, z est un réel ; si x = 0, z est un imaginaire pur.

Égalité de deux complexes : x + i y = x' + i y' ⇔ x = x' et y = y'.

Plan complexe : on munit le plan d'un repère orthonormal direct (O; u, v). À tout complexe z = x + i y, on associe le point M(x,y) ou le vecteur OM. On dit que z est l'affixe de M.

2. Opérations algébriques

2.1 Addition et soustraction

(x + i y) + (x' + i y') = (x+x') + i(y+y'). L'addition correspond à l'addition vectorielle dans le plan.

2.2 Multiplication

(x + i y)(x' + i y') = x x' + i x y' + i y x' + i² y y' = (x x' – y y') + i (x y' + y x').

2.3 Conjugaison

Le conjugué de z = x + i y est z = x – i y. Il vérifie :

  • z = z
  • z + z = 2 Re(z) ; z – z = 2i Im(z)
  • z · z = x² + y² = |z|² (nombre réel positif)
  • Le conjugué d'une somme/produit/quotient est la somme/produit/quotient des conjugués.

Exercice 1 : Soit z₁ = 3 + 2i, z₂ = 1 – i. Calculer z₁·z₂, z₁/z₂, et vérifier que z₁·z₂ = z₁ · z₂.

z₁·z₂ = 3×1 + 3×(–i) + 2i×1 + 2i×(–i) = 3 –3i +2i –2i² = 3 – i + 2 = 5 – i.
z₁/z₂ = (3+2i)(1+i)/(1²+1²) = (3+3i+2i+2i²)/2 = (3+5i–2)/2 = (1+5i)/2.
Conjugués : z₁=3–2i, z₂=1+i. Leur produit = (3–2i)(1+i)=3+3i–2i–2i² = 3+i+2=5+i = 5–i (car 5–i est bien le conjugué de 5+i). Vérifié.

3. Module d'un nombre complexe

Définition : le module de z = x + i y est le réel positif |z| = √(x² + y²). Géométriquement, c'est la distance OM.

Propriétés :

  • |z| ≥ 0, |z| = 0 ⇔ z = 0
  • |z| = |z|
  • |z·z'| = |z|·|z'|, |z/z'| = |z|/|z'| (si z'≠0)
  • Inégalité triangulaire : |z + z'| ≤ |z| + |z'|
  • ||z| – |z'|| ≤ |z – z'|

Exercice 2 : Déterminer l'ensemble des points M d'affixe z tels que |z – 2 + i| = 3.

|z – (2 – i)| = 3 ⇔ distance de M au point A(2–i) est constante égale à 3. C'est le cercle de centre A et de rayon 3.

4. Argument et forme trigonométrique

Pour z ≠ 0, un argument de z, noté arg(z), est une mesure de l'angle orienté (u, OM). Il est défini modulo 2π. On a :

cosθ = x/|z|, sinθ = y/|z|.

Forme trigonométrique : z = r (cosθ + i sinθ) avec r = |z|, θ = arg(z) [2π].

Si z = 0, le module est 0 mais l'argument n'est pas défini.

4.1 Propriétés des arguments

  • arg(z·z') = arg(z) + arg(z') [2π]
  • arg(z/z') = arg(z) – arg(z') [2π]
  • arg(zn) = n arg(z) [2π] (n entier)
  • arg(z) = – arg(z) [2π]

Exercice 3 : Mettre sous forme trigonométrique z = –1 + i√3.

Module : |z| = √(1+3) = 2. cosθ = –1/2, sinθ = √3/2 ⇒ θ = 2π/3.
Donc z = 2 (cos(2π/3) + i sin(2π/3)).

5. Notation exponentielle et formules d'Euler

On pose eiθ = cosθ + i sinθ. Alors tout complexe non nul s'écrit z = r eiθ (forme exponentielle).
Cette notation simplifie les calculs :

  • r eiθ · r' eiθ' = (rr') ei(θ+θ')
  • 1/(r eiθ) = (1/r) e–iθ
  • (r eiθ)n = rn ei nθ (Formule de Moivre)

5.1 Formules d'Euler inverses

cosθ = (eiθ + e–iθ)/2, sinθ = (eiθ – e–iθ)/(2i).

5.2 Linéarisation

Ces formules permettent d'exprimer des puissances de cosinus et sinus en sommes de cos(nθ). Exemple classique : linéariser sin³θ.

Exercice 4 : Linéariser cos⁴θ.

cos⁴θ = [ (eiθ+e–iθ)/2 ]⁴ = (1/16)(e4iθ + 4 e2iθ + 6 + 4 e–2iθ + e–4iθ) = (1/8)(cos(4θ) + 4 cos(2θ) + 3).

6. Racines n-ièmes d'un nombre complexe

6.1 Racines carrées

Pour déterminer les racines carrées de Z = a+ib, on résout (x+iy)² = a+ib, soit le système :

x² – y² = a, 2xy = b, avec x² + y² = |Z|.

On obtient x² = (|Z|+a)/2, y² = (|Z|–a)/2, et le signe de xy donné par b. Chaque nombre complexe non nul admet deux racines carrées opposées.

6.2 Racines n-ièmes de l'unité

Les solutions de zn = 1 sont les racines n-ièmes de l'unité : ωk = e2i kπ/n, k = 0, 1, …, n–1. Elles forment un polygone régulier à n côtés sur le cercle unité.

Propriété : somme des racines n-ièmes de l'unité = 0.

Exercice 5 : Résoudre z³ = –8.

–8 = 8 eiπ (ou 8 ei(π+2kπ)). Les solutions sont zk = 2 ei(π+2kπ)/3 pour k=0,1,2.
k=0 : 2 eiπ/3 = 2(1/2 + i√3/2) = 1 + i√3.
k=1 : 2 eiπ = –2.
k=2 : 2 ei5π/3 = 2(1/2 – i√3/2) = 1 – i√3.

7. Équations du second degré dans ℂ

7.1 Coefficients réels

Soit az² + bz + c = 0, a,b,c réels, a≠0. Discriminant Δ = b² – 4ac.
Si Δ ≥ 0 : solutions réelles connues.
Si Δ < 0 : deux solutions complexes conjuguées : z = (–b ± i√|Δ|)/(2a).

7.2 Coefficients complexes

Si a,b,c sont quelconques dans ℂ, on utilise la même méthode avec Δ complexe, en cherchant une racine carrée δ de Δ, puis les solutions sont (–b ± δ)/(2a).

Exercice 6 : Résoudre z² – (3+2i)z + (1+3i) = 0.

Δ = (3+2i)² – 4(1+3i) = 9 + 12i + 4i² – 4 – 12i = (9–4–4) + i(12–12) = 1. Donc δ = 1 (ou –1).
Solutions : z₁ = (3+2i + 1)/2 = 2 + i ; z₂ = (3+2i – 1)/2 = 1 + i.

8. Nombres complexes et géométrie plane

8.1 Affixes de points, vecteurs

Soient A(zA), B(zB). Le vecteur AB a pour affixe zB – zA. Le milieu I de [AB] a pour affixe (zA+zB)/2.

Distance AB = |zB – zA|. Angle orienté (u, AB) = arg(zB – zA) [2π].

8.2 Transformations

  • Translation de vecteur w d'affixe t : M(z) → M'(z+t).
  • Homothétie de centre Ω(ω), rapport k (réel non nul) : M(z) → M'(ω + k(z–ω)).
  • Rotation de centre Ω(ω), angle θ : M(z) → M'(ω + eiθ(z–ω)).
  • Similitude directe de centre Ω, rapport k>0, angle θ : z' – ω = k eiθ (z – ω).

Exercice 7 : Soit A(1), B(2+i). Déterminer l'affixe de C tel que le triangle ABC soit équilatéral direct.

Un triangle équilatéral direct se construit par la rotation de centre A et d'angle π/3 appliquée à B. Soit zC – zA = eiπ/3 (zB – zA).
zA = 1, zB – zA = 1+i. eiπ/3 = 1/2 + i√3/2.
Donc zC = 1 + (1/2 + i√3/2)(1+i) = 1 + (1/2)(1+i) + i√3/2(1+i) = ... = (1 + 1/2 – √3/2) + i(1/2 + √3/2) = (3 – √3)/2 + i(1+√3)/2.

9. Caractérisation d'ensembles de points

On utilise les interprétations du module et de l'argument.

  • Cercle : |z – a| = r.
  • Médiatrice : |z – a| = |z – b|.
  • Demi-droite : arg(z – a) = θ [2π] (demi-droite ouverte d'origine A).
  • Droite : arg(z – a) = θ [π] (droite passant par A privée de A).

Exercice 8 : Déterminer l'ensemble des points M d'affixe z tels que |z – 1| = |z – i|.

C'est la médiatrice du segment [AB] avec A(1) et B(i). Cette médiatrice est une droite d'équation cartésienne x = y (car A et B symétriques par rapport à cette droite).

10. Exercices supplémentaires

Exercice 9 : On pose z = (√3 – i)/(1 – i√3). Donner la forme algébrique, le module, un argument, et en déduire cos(π/12) et sin(π/12).

Simplifions : z = (√3–i)(1+i√3)/(1+3) = ( √3 + 3i – i – i²√3 ) /4 = ( √3 + 2i + √3 ) /4 = (2√3 + 2i)/4 = (√3 + i)/2.
Module : |z| = √(3+1)/2 = 1. Argument : cosθ = √3/2, sinθ = 1/2 ⇒ θ = π/6. Donc z = eiπ/6 = cos(π/6)+i sin(π/6).
En développant la forme initiale, on obtient aussi ei(π/3 – π/4) = eiπ/12. Conclusion : cos(π/12) = √3/2, sin(π/12) = 1/2 ? Attention, erreur : en réalité, l'angle trouvé est π/6, pas π/12. Rectification : le module est 1, argument π/6, donc pas π/12. (Cet exercice nécessite de revoir le calcul).
