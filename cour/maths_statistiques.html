1. Statistique descriptive univariée

La statistique descriptive a pour but de résumer, présenter et analyser un ensemble de données. On distingue les caractères qualitatifs et quantitatifs (discrets ou continus).

1.1 Effectifs, fréquences, tableaux

Pour un caractère prenant les valeurs x1, x2, …, xp avec effectifs n1, …, np (total N), la fréquence est fi = ni/N. On peut aussi calculer les effectifs cumulés croissants (ECC) et les fréquences cumulées croissantes (FCC).

1.2 Indicateurs de position

Moyenne : x = (1/N) Σ ni xi.
Médiane Me : valeur qui partage la série en deux effectifs égaux (au moins 50% des valeurs ≤ Me et ≥ Me). Pour une série discrète, on utilise les effectifs cumulés.
Mode : valeur la plus fréquente.

1.3 Indicateurs de dispersion

Étendue = max – min.
Variance V = (1/N) Σ ni (xi – x)² = (1/N) Σ ni xi² – x².
Écart-type σ = √V (ou s pour un échantillon).
Intervalle interquartile = Q3 – Q1 (où Q1 et Q3 sont les premier et troisième quartiles).

1.4 Diagramme en boîte (boxplot)

Il représente : min, Q1, médiane, Q3, max. Les moustaches s'arrêtent généralement à 1.5×(Q3–Q1) au-delà desquelles les points sont atypiques.

N = 3+5+7+4+1 = 20. Moyenne = (8×3+10×5+12×7+14×4+16×1)/20 = (24+50+84+56+16)/20 = 230/20 = 11.5.
Variance = [(24×8)+(50×10)+(84×12)+(56×14)+(16×16)]/20 – 11.5² = (192+500+1008+784+256)/20 – 132.25 = 2740/20 – 132.25 = 137 – 132.25 = 4.75. Écart-type ≈ 2.18.
Médiane : 10e et 11e valeurs : les deux sont 12 (car ECC : 3,8,15…). Me=12. Q1 : 5e valeur = 10 ; Q3 : 15e valeur = 14.

2. Statistique bivariée : étude de la corrélation

On observe deux variables quantitatives x et y sur n individus. On obtient une série double (xi, yi). Le nuage de points permet de visualiser une éventuelle relation.

2.1 Point moyen

Le point moyen est G(x, y) où x et y sont les moyennes respectives. Il est le centre de gravité du nuage.

2.2 Covariance

Cov(x,y) = (1/n) Σ (xi – x)(yi – y) = (1/n) Σ xiyi – x y.

2.3 Coefficient de corrélation linéaire r

r = Cov(x,y) / (σx σy). –1 ≤ r ≤ 1. Si |r| proche de 1, la liaison linéaire est forte. Si r proche de 0, il n'y a pas de corrélation linéaire.

Exercice 2 : Calculer r pour les données suivantes :
x : 1, 2, 3, 4, 5 ; y : 2, 4, 6, 8, 10.

Moyennes : x̄=3, ȳ=6. Cov = [(1-3)(2-6)+(2-3)(4-6)+…]/5 = [(-2)(-4)+(-1)(-2)+0+ (1)(2)+(2)(4)]/5 = [8+2+0+2+8]/5=20/5=4. σx²= (4+1+0+1+4)/5=2, σx=√2. σy²= (16+4+0+4+16)/5=8, σy=√8. r = 4/(√2×√8) = 4/√16 = 4/4 = 1. Corrélation parfaite.

3. Ajustement affine par la méthode des moindres carrés

On cherche la droite d'équation y = a x + b minimisant la somme des carrés des écarts verticaux Σ (yi – (a xi + b))². Les coefficients sont :

a = Cov(x,y) / V(x)
b = y – a x.

La droite de régression de y en x passe par le point moyen G. On peut aussi définir la droite de régression de x en y si on souhaite prédire x à partir de y.

3.1 Qualité de l'ajustement : coefficient de détermination R² = r²

Plus R² est proche de 1, meilleur est l'ajustement. R² indique la part de variance de y expliquée par x.

3.2 Interpolation et extrapolation

La droite permet d'estimer une valeur de y pour un x compris dans la plage (interpolation) ou en dehors (extrapolation, à utiliser avec prudence).

Exercice 3 : On a relevé la température (x en °C) et la consommation électrique (y en MW) :
(15, 200), (10, 220), (5, 240), (0, 260). Déterminer la droite de régression de y en x, puis estimer la consommation pour 8°C.

Moyennes : x̄= (15+10+5+0)/4=7.5, ȳ= (200+220+240+260)/4=230. Cov = [(7.5)(–30)+(2.5)(–10)+(–2.5)(10)+(–7.5)(30)]/4 = [(–225)+(–25)+(–25)+(–225)]/4 = –500/4 = –125. V(x)= (7.5²+2.5²+2.5²+7.5²)/4 = (56.25+6.25+6.25+56.25)/4 = 125/4=31.25. a = –125/31.25 = –4. b = 230 – (–4)×7.5 = 230+30=260. Droite : y = –4x + 260. Pour x=8, y = –32+260=228 MW.

4. Ajustements non linéaires

Lorsque le nuage n'est pas rectiligne, on peut effectuer un changement de variable pour se ramener à une forme linéaire. Exemples classiques :

  • Modèle exponentiel y = A ek x → ln y = ln A + k x. On ajuste (x, ln y).
  • Modèle puissance y = B xa → ln y = ln B + a ln x. On ajuste (ln x, ln y).
  • Modèle inverse y = a + b/x → poser t = 1/x.

Exercice 4 : On mesure une population bactérienne : t (heures) : 0, 1, 2, 3 ; N (milliers) : 1, 2, 4, 8. Ajuster un modèle exponentiel N = N₀ eλt.

ln N : t=0→0 ; t=1→ln2≈0.693 ; t=2→ln4≈1.386 ; t=3→ln8≈2.079. On ajuste (t, ln N). Moyennes : t̄=1.5, lnN̄≈1.04. Cov(t,lnN)=? Ajustement linéaire de lnN sur t donne ln N = λ t + ln N₀. Avec points alignés, λ = ln2 ≈0.693, N₀=1. Modèle parfait N = e0.693t.

5. Séries chronologiques

Une série chronologique est une suite de valeurs observées à intervalles réguliers (mois, années). On peut étudier la tendance (moyennes mobiles), la saisonnalité et effectuer une prévision.

5.1 Moyenne mobile

Pour lisser la série, on remplace chaque valeur par la moyenne d'un nombre fixe de valeurs autour (ex : moyenne mobile d'ordre 3).

5.2 Coefficient saisonnier

On calcule la tendance, puis on détermine un coefficient saisonnier = valeur / tendance. Ensuite, on prévoit en combinant tendance projetée et saisonnalité.

Exercice 5 : Ventes trimestrielles (en milliers) : T1: 100, T2: 150, T3: 120, T4: 180, T1: 110, T2: 160… Calculer les moyennes mobiles centrées d'ordre 4.

Moyenne mobile d'ordre 4 : (100+150+120+180)/4 = 137.5, etc. On centre ensuite entre deux trimestres. (Calculs détaillés selon données).
