1. Limites et continuité des fonctions

La notion de limite est à la base de l'analyse. Elle permet de décrire le comportement d'une fonction lorsque la variable s'approche d'une valeur donnée ou devient très grande. La continuité en découle directement.

1.1 Approche intuitive et définitions formelles

1.1.1 Limite finie en un point fini

Soit a ∈ ℝ et L ∈ ℝ. On dit que limx→a f(x) = L si pour tout intervalle ouvert contenant L (aussi petit soit-il), il existe un intervalle ouvert contenant a tel que pour tout x ≠ a dans cet intervalle, f(x) appartient au premier intervalle. Graphiquement, la courbe se rapproche du point (a, L).

Exemple fondamental : f(x) = (x² – 1)/(x – 1) pour x ≠ 1. En simplifiant, f(x) = x+1, donc limx→1 f(x) = 2, bien que f ne soit pas définie en 1.

1.1.2 Limite infinie en un point

limx→a f(x) = +∞ signifie que f(x) dépasse toute valeur donnée dès que x est assez proche de a. De même pour –∞. Exemple : limx→0⁺ 1/x = +∞, limx→0⁻ 1/x = –∞.

1.1.3 Limite finie à l'infini

limx→+∞ f(x) = L si la distance entre f(x) et L devient aussi petite que l'on veut pour x suffisamment grand. Exemple : limx→+∞ 1/x = 0.

1.1.4 Limite infinie à l'infini

limx→+∞ f(x) = +∞ lorsque f(x) devient arbitrairement grande avec x. Exemple : limx→+∞ x² = +∞.

1.3 Techniques de levée d'indétermination

  • Factorisation : On met en facteur le terme de plus haut degré. Exemple : pour limx→+∞ (3x² – x + 1)/(x – 2), on factorise x² au numérateur et x au dénominateur.
  • Quantité conjuguée : Pour les expressions avec des racines carrées. Exemple : (√(x+1) – 1)/x.
  • Utilisation des limites de référence et croissances comparées.
  • Règle de l'Hôpital (hors programme) : non exigible, mais peut être évoquée.

1.5 Théorèmes de comparaison

Théorème des gendarmes : Si pour x assez grand, u(x) ≤ f(x) ≤ v(x) et lim u = lim v = L (fini), alors lim f = L.
Extension à l'infini : Si u(x) ≤ f(x) et lim u = +∞, alors lim f = +∞.
Utilisation de la valeur absolue : Si |f(x) – L| ≤ g(x) et lim g(x) = 0, alors lim f(x) = L.

1.6 Continuité

Définition : f est continue en a si limx→a f(x) = f(a). Graphiquement, pas de saut. Les fonctions usuelles (polynômes, rationnelles, racines, trigonométriques, exponentielles, logarithmes) sont continues sur leur domaine de définition.

Prolongement par continuité : Si f n'est pas définie en a mais admet une limite finie L, on peut la prolonger en posant g(a) = L.

1.7 Théorème des valeurs intermédiaires (TVI)

Énoncé : Soit f continue sur un intervalle I, a, b dans I, a < b. Pour tout réel k compris entre f(a) et f(b), il existe au moins un c ∈ [a, b] tel que f(c) = k.
Corollaire pour résolution d'équation : Si f(a) × f(b) < 0, alors l'équation f(x) = 0 admet au moins une solution dans ]a, b[.

Méthode de dichotomie : On partage l'intervalle en deux et on conserve la moitié où f change de signe, jusqu'à la précision voulue.

Exercice 1 : Déterminer limx→+∞ (√(x² + 3x) – x).

Forme ∞ – ∞. On utilise la quantité conjuguée :
√(x² + 3x) – x = ( (x²+3x) – x² ) / (√(x²+3x) + x) = 3x / (√(x²+3x) + x).
On factorise x : = 3 / (√(1 + 3/x) + 1). Quand x→+∞, √(1+3/x) → 1, donc la limite est 3/2.

Exercice 2 : Montrer que l'équation ex + x – 2 = 0 admet une unique solution dans ℝ.

f(x)=ex+x–2 est strictement croissante (dérivée ex+1>0). f(0)= –1, f(1)=e–1>0. D'après le TVI, il existe un unique c∈]0,1[ tel que f(c)=0.

2. Dérivation

La dérivée mesure la vitesse de variation d'une fonction. Elle permet de déterminer les tangentes, les variations, les extremums.

2.1 Nombre dérivé et fonction dérivée

Définition : f est dérivable en a si la limite limh→0 (f(a+h) – f(a))/h existe et est finie. Cette limite est notée f '(a). Géométriquement, c'est le coefficient directeur de la tangente au point (a, f(a)).

La fonction dérivée f ' est définie là où la limite existe.

2.3 Règles de dérivation

  • (u + v)' = u' + v'
  • (k u)' = k u'
  • (u v)' = u' v + u v'
  • (u/v)' = (u'v – u v') / v² (v non nulle)
  • (un)' = n u' un–1
  • Fonction composée : (g ∘ f)'(x) = g'(f(x)) × f'(x)

Exemple de composée : f(x) = e2x+1, f '(x) = 2 e2x+1. Ou f(x) = ln(3x+1), f '(x) = 3/(3x+1).

2.4 Équation de la tangente

Tangente au point d'abscisse a : y = f '(a)(x – a) + f(a).

2.5 Lien entre signe de la dérivée et variations

Si f '(x) ≥ 0 sur un intervalle, f est croissante ; si f '(x) ≤ 0, f est décroissante. Si f ' s'annule en changeant de signe, on a un extremum local.

Tableau de variations : on étudie le signe de f ' et on en déduit les flèches.

2.6 Théorèmes de Rolle et des accroissements finis

Rolle : f continue sur [a,b], dérivable sur ]a,b[, f(a)=f(b) ⇒ ∃ c∈]a,b[ avec f '(c)=0.

Accroissements finis : ∃ c∈]a,b[ tel que f(b) – f(a) = f '(c) (b – a).

2.7 Dérivées successives

On peut dériver plusieurs fois : f '', f '''… La dérivée seconde donne des informations sur la convexité. Si f ''(x) ≥ 0, la fonction est convexe (sa courbe est au-dessus de ses tangentes).

Exercice 3 : Soit f(x) = (x²+1)/(x–1). Déterminer le domaine, f '(x), et les variations.

Domaine : ℝ \ {1}. f '(x) = [2x(x–1) – (x²+1)]/(x–1)² = (x² – 2x – 1)/(x–1)². Signe : le dénominateur >0, le numérateur a pour racines 1±√2. Tableau : croissante sur ]–∞, 1–√2] et [1+√2,+∞[, décroissante sur [1–√2,1[ et ]1,1+√2].

Exercice 4 : Montrer que pour tout x > –1, ln(1+x) ≤ x.

Soit g(x) = ln(1+x) – x. g'(x) = 1/(1+x) – 1 = –x/(1+x). Sur ]–1,0[, g' >0 ; sur ]0,+∞[, g' <0. g admet un maximum en 0 : g(0)=0, donc g(x) ≤ 0 pour tout x, d'où l'inégalité.

Exercice 5 : Application économique (si taux d'accroissement). Mais on peut proposer un exercice sur la tangente : Déterminer les points de la courbe y = x³ – 3x où la tangente est horizontale.

Tangente horizontale ⇔ f '(x)=0. f '(x)=3x²–3=3(x–1)(x+1). Donc x = –1 et x = 1. Points : (–1, 2) et (1, –2).

3. Fonction logarithme népérien et exponentielle

Le logarithme népérien et l'exponentielle sont deux fonctions réciproques fondamentales. Elles apparaissent dans les modèles de croissance, décroissance, et dans la résolution d'équations.

3.1 Logarithme népérien (ln)

Définition : La fonction ln est l'unique primitive de x ↦ 1/x sur ]0,+∞[ qui s'annule en 1. Elle est strictement croissante, continue, dérivable, ln'(x) = 1/x. ln 1 = 0, ln e = 1.

Propriétés algébriques : Pour a, b > 0, ln(ab) = ln a + ln b, ln(a/b) = ln a – ln b, ln(an) = n ln a (n réel).

3.1.1 Limites et croissance comparée

  • limx→0⁺ ln x = –∞, limx→+∞ ln x = +∞
  • limx→+∞ (ln x)/xα = 0 (α > 0) : « les puissances l'emportent sur le logarithme ».
  • limx→0⁺ xα ln x = 0.

3.2 Fonction exponentielle (ex)

Définition : La fonction exponentielle est la réciproque de ln : pour tout x réel, y = ex ⇔ x = ln y (y>0). Elle est strictement croissante, dérivable, (ex)' = ex. e0 = 1.

Propriétés : ea+b = ea eb, ea–b = ea/eb, (ea)n = ena, eln x = x, ln(ex) = x.

3.2.1 Limites et croissance comparée

  • limx→+∞ ex = +∞, limx→–∞ ex = 0
  • limx→+∞ ex/xα = +∞ (α > 0) : « l'exponentielle l'emporte sur les puissances ».
  • limx→–∞ |x|α ex = 0.

3.3 Résolution d'équations et inéquations

Utiliser la bijectivité : ln A = ln B ⇔ A = B (A,B>0) ; eA = eB ⇔ A = B. Inéquations : ln A < ln B ⇔ A < B ; eA < eB ⇔ A < B.

Exercice 6 : Résoudre e2x – 3 ex – 4 = 0.

Poser X = ex > 0. X² – 3X – 4 = 0 → X = 4 ou X = –1 (rejeté). Donc ex = 4 → x = ln 4 = 2 ln 2.

Exercice 7 : Étudier la fonction f(x) = x ln x – x et dresser son tableau de variations.

Domaine ]0,+∞[. f '(x) = ln x + 1 – 1 = ln x. f '(x) = 0 pour x=1. f ' négative sur ]0,1[, positive sur ]1,+∞[. f décroît puis croît. Minimum f(1) = –1. Limites : en 0⁺, f(x) → 0 (car x ln x →0) ; en +∞, f(x) → +∞.

Exercice 8 : Déterminer les limites aux bornes de f(x) = ln(x+1) / x.

Domaine ]–1,0[∪]0,+∞[. En 0, forme 0/0 : on reconnaît le nombre dérivé de ln(1+x) en 0, limite = 1. En –1⁺, ln(0⁺)= –∞, donc limite –∞. En +∞, par croissance comparée, (ln x)/x →0, donc ici pareil : 0.
