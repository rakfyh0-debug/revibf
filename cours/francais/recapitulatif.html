1.3 Les types et formes de phrases

  • Types : déclaratif (.), interrogatif (?), impératif/injonctif (! ou .), exclamatif (!).
  • Formes : affirmative / négative (ne…pas, ne…plus…), active / passive, neutre / emphatique.

1.4 Les propositions

  • Indépendante : ne dépend d'aucune autre. Le soleil brille.
  • Principale : commande une subordonnée. Je pense que tu as raison.
  • Subordonnée relative : introduite par qui, que, dont, oà¹, complète un nom. Le livre que j'ai lu…
  • Subordonnée conjonctive : introduite par que, quand, parce que…, fonction de COD, CC… Je crois qu'il viendra.
  • Subordonnée interrogative indirecte : introduite par si, ce que, pourquoi… Il demande si tu viens.

2. CONJUGAISON

2.1 Les groupes de verbes

  • 1er groupe : verbes en -er (sauf aller). Chanter, aimer.
  • 2e groupe : verbes en -ir avec participe présent en -issant. Finir (finissant), choisir.
  • 3e groupe : tous les autres (verbes en -re, -oir, -ir sans -issant, aller…). Prendre, pouvoir, venir, aller.

2.2 Les modes et les temps

  • Indicatif (mode du réel) : présent, imparfait, passé simple, futur simple, passé composé, plus‑que‑parfait, futur antérieur.
  • Conditionnel (hypothèse, souhait) : présent, passé.
  • Subjonctif (doute, obligation, souhait) : présent, passé.
  • Impératif (ordre, conseil) : présent.
  • Infinitif et Participe (présent, passé).

2.3 Les valeurs des temps (à connaà®tre pour le BEPC)

  • Présent de l'indicatif : action en cours, vérité générale, habitude, passé récent, futur proche.
  • Imparfait : action passée en train de se dérouler, description, habitude passée.
  • Passé simple : action brève et achevée dans un récit au passé (employé avec l'imparfait).
  • Futur simple : action à venir.
  • Conditionnel présent : hypothèse (si + imparfait → conditionnel), souhait, politesse, information non confirmée.
  • Subjonctif présent : après certaines conjonctions (pour que, avant que, bien que…), pour exprimer le souhait ou l'obligation.

3. ORTHOGRAPHE

3.1 Les règles d'accord

  • Accord du verbe avec le sujet : le verbe s'accorde en personne et en nombre avec son sujet (màªme si le sujet est éloigné ou inversé).
  • Accord dans le groupe nominal : le déterminant, l'adjectif qualificatif, le participe passé employé comme adjectif s'accordent en genre et en nombre avec le nom qu'ils accompagnent.
  • Accord du participe passé :
    • avec àªtre → accord avec le sujet. Elles sont arrivées.
    • avec avoir → accord avec le COD si celui-ci est placé avant le verbe. Les fleurs que j'ai cueillies.
    • sans auxiliaire → employé comme adjectif, il s'accorde avec le nom. Des feuilles mortes.

3.3 Distinguer l'infinitif en -er du participe passé en -é

Remplacer par un verbe du 3e groupe (vendre, prendre). Si on peut mettre vendu, c'est -é ; si on peut mettre vendre, c'est -er.

Exemple : Il a mangé → Il a vendu. Donc -é.
Il veut manger → Il veut vendre. Donc -er.

4. VOCABULAIRE

4.1 Formation des mots

  • Radical : partie qui porte le sens principal. Chant- (chanter, chanteur).
  • Préfixe : élément placé avant le radical. re-, in-, dé-.
  • Suffixe : élément placé après le radical. -eur, -able, -ment, -tion.
  • Famille de mots : ensemble de mots ayant le màªme radical. Jour, journal, journée, ajourner.

4.2 Sens des mots

  • Synonymes : mots de màªme sens ou de sens proche. Heureux / content.
  • Antonymes : mots de sens contraire. Grand / petit.
  • Homonymes : mots qui se prononcent de la màªme façon mais ont un sens différent. Ver / vers / vert.
  • Polysémie : un mot peut avoir plusieurs sens selon le contexte. Une feuille (de papier, d'arbre).
  • Sens propre / sens figuré : sens réel / sens imagé. La tàªte d'un homme / àªtre à la tàªte de l'entreprise.

4.3 Les niveaux de langue

  • Soutenu : littéraire, très correct. Je vous prie de bien vouloir m'excuser.
  • Courant : utilisé dans la vie de tous les jours. Excusez-moi.
  • Familier : relâché, entre amis. Désolé !

5. EXPRESSION ÉCRITE

5.1 Les principaux types de textes

  • Texte narratif : raconte une histoire. Organisé en situation initiale, élément perturbateur, péripéties, résolution, situation finale.
  • Texte descriptif : décrit un lieu, un personnage, un objet. Organisé dans l'espace.
  • Texte argumentatif : défend une opinion. Structure : introduction, développement (arguments + exemples), conclusion.
  • Texte explicatif : explique un phénomène. Souvent objectif, avec des définitions et des exemples.
  • Texte injonctif : donne des consignes, des ordres (recette, notice).
  • La lettre : personnelle (amicale) ou administrative (formelle).

5.2 La structure d'une rédaction argumentative

  1. Introduction : amener le sujet, poser la problématique, annoncer le plan.
  2. Développement : 2 à 3 paragraphes, chacun avec un argument et un exemple.
  3. Conclusion : résumer la position, élargir éventuellement le débat.

5.4 Conseils pour réussir une rédaction

  • Analyser attentivement le sujet (type de texte, destinataire, longueur).
  • Chercher des idées au brouillon (liste, schéma).
  • Organiser les idées dans un plan.
  • Rédiger en soignant la ponctuation, la grammaire et l'orthographe.
  • Relire pour corriger les fautes.

6. LECTURE ET COMPRÉHENSION

6.1 Les genres littéraires

  • Le roman : récit long en prose.
  • La nouvelle : récit court centré sur un seul événement.
  • Le conte : récit merveilleux avec une morale.
  • La poésie : texte en vers, rimes, rythme.
  • Le théâtre : texte destiné à àªtre joué (dialogues, didascalies).
  • La bande dessinée : succession d'images avec texte dans des bulles.
  • Les textes documentaires : article de journal, essai, biographie.

6.2 Les figures de style courantes

  • Comparaison : « Il est rusé comme un renard. »
  • Métaphore : « Cet homme est un roc. »
  • Personnification : « Le vent hurle. »
  • Hyperbole : « Je meurs de faim. »
  • Anaphore : répétition en début de phrase. « Paris ! Paris outragé ! Paris brisé ! »
  • Gradation : « Va, cours, vole ! »
  • Antithèse : opposition. « Je vis, je meurs. »

6.3 Méthode pour analyser un texte

  1. Lire le texte en entier.
  2. Identifier le thème (de quoi parle le texte ?).
  3. Repérer le type de texte (narratif, argumentatif…).
  4. Dégager les idées principales de chaque paragraphe.
  5. Analyser les procédés d'écriture (vocabulaire, figures de style, temps des verbes).
  6. Comprendre l'intention de l'auteur (informer, convaincre, émouvoir…).
